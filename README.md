spring-and-angular
==================

A SPA demo project with Spring Framework and Angularjs

Make sure mimosa is installed:
* `npm install -g mimosa`

Make sure karma is installed
* `npm install -g karma karma-cli`

front end is under the frontend/ folder:
* `npm install`
* `mimosa build`

to run karma tests:
* `karma start`

to run backend:
* `mvn spring-boot:run`
